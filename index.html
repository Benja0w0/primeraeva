Concepto de HTML5:

   Html5 es la quinta versión de Html, lo único que agrega son cosas nuevas como etiquetas y funciones, pero su función principal sigue siendo la misma que cuando se creó, y esa es poder diseñar páginas web mucho más fácil, lo triste de esto es que por sí solo una página hecha con HTML se ve muy mal, así que se debe completar con otro lenguaje para poder darle un toque visual más estético. Los lenguajes que sirven para complementarlo serian CSS o JavaScript, por ejemplo, esta página fue complementada con CSS para dar un apartado grafico más decente.

Caracteristicas:

  A continuación veremos sus características y una breve explicación de ellas.

  • Semántica: Permite describir con mayor precisión cuál es su contenido.

  • Conectividad: Permite comunicarse con el servidor de formas nuevas e innovadoras.

  • Sin conexión y almacenamiento: Permite a las páginas web almacenar datos localmente en el lado del cliente y operar sin conexión de manera más eficiente

  • Multimedia: Nos otorga un excelente soporte para utilizar contenido multimedia como lo son audio y video nativamente.

  • Gráficos y efectos 2D/3D: Proporciona una amplia gama de nuevas características que se ocupan de los gráficos en la web como lo son canvas 2D, WebGL, SVG, etc.

  • Rendimiento e Integración: Proporciona una mayor optimización de la velocidad y un mejor uso del hardware.

  • Acceso al dispositivo: Proporciona APIs para el uso de varios componentes internos de entrada y salida de nuestro dispositivo.

  • CSS3: Nos ofrece una nueva gran variedad de opciones para hacer diseños más sofisticados.

Paginas de referencias:

  • Etiquetas de texto: este tipo de etiquetas en html nos sirven para poder modificar el texto que queremos que le aparezca al usuario, ya sea la información que queremos que el vea o contenido segundario que le podría interesar, sin embargo, es importante saberla clasificar ya que los navegadores siempre descartan las páginas que tienen una mala sintaxis en sus búsquedas. Si desea saber más sobre las etiquetas de contenido y texto, dele click aquí.

  • Etiquetas de encabezados , ubicación y formato: son las que nos permite jugar con el texto, a diferencia de las anteriores que nos permiten clasificar el texto, estas nos permiten modificar sus características, pudiendo resaltarlas, hacerlas más grandes o incluso modificar la fuente sin tener que usar CSS o JavaScrip, sin embargo también hay una que otra etiqueta que nos permite clasificar el texto además de modificarlo (por ejemplo, los encabezados).   • Etiquetas para numeración y viñetas: estas nos sirven para crear las “listas” que nos permitirán también organizar nuestra información, pero de una manera más simple y sencilla, click aquí. para verlas.
